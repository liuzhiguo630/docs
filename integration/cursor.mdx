官网地址: [https://www.cursor.com/cn](https://www.cursor.com/cn)

## 准备：


**API Key**：您的登陆码

**OpenAI Base URL**：https://ai.liaobots.work/v1

## 流程：

<img src="/images/cursor/step-1.jpg" />

1.点击设置

<img src="/images/cursor/step-2.jpg" />

2.在左侧 「Models」 里下滑找到 「OpenAI API Key」 并输入 「API Key」 与 「OpenAI Base URL」

3.设置完成后点击 Verify 后无报错即为成功连接

<Warning>
对于非 OpenAI 模型需更改模型名并添加使用
</Warning>

<Card title="添加模型" img="/images/cursor/tips.jpg">
    先点击 「Add Model」 ，且添加模型需在模型名前加 「liao-」
    例如 「grok-3」 应改为 「liao-grok-3」
</Card>
