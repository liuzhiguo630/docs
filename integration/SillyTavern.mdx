## 准备：


**自定义端点**：https://ai.liaobots.work/v1

**自定义 API 密钥**：您的登陆码

## 流程：

点击顶部插头图标-输入 「自定义端点」「自定义 API 密钥」-点击 「连接」-点击 「发送测试消息」

模型名：在「模型与价格中」找到您需要调用的模型名并复制下来

<img src="/images/SillyTavern/step-1.jpg" />

1.点击顶部插头状图标填入 「自定义端点」「自定义 API 密钥」「模型名」

<img src="/images/SillyTavern/step-2.jpg" />

2.点击左下角 「连接」 按钮后再点击 「发送测试消息」
