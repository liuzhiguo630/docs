## 准备：

模型名称：在「模型与价格中」找到您需要调用的模型名并复制下来

API Key：您的登陆码

API endpoint URL：https://ai.liaobots.work/v1

## 流程：

点击设置-点击 「模型供应商」-选择 「OpenAl-API-compatible」 -

点击 「添加模型」-输入 「模型名」「API endpoint URL」「API Key」 后点击 「保存」
