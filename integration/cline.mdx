在 VScode 里搜索 cline 并安装

## 准备：

Base URL：https://ai.liaobots.work/v1

API Key：您的登陆码

Model ID：在「模型与价格中」找到您需要调用的模型名并复制下来

## 流程：

Use your own API Key-API Provider-OpenAI Compatible-

输入 「Base URL」「API Key」「Model ID」

<img src="/images/cline/step-1.jpg" />

1.选择 「Use your own API Key」

<img src="/images/cline/step-2.jpg" />

2.在 「API Provider」 里选择 「OpenAI Compatible」

<img src="/images/cline/step-3.jpg" />

3.填入 「BASE URL」「API Key」以及 「Model ID」

<img src="/images/cline/step-4.jpg" />

4.在右下角点击 「plan」 后发送测试消息
