Install Cursor

Official website: [https://www.cursor.com/cn](https://www.cursor.com/cn)

## Preparation

> `OpenAI Base URL`：https://ai.liaobots.work/v1
>
>
> `API Key`：Your Login Code
>
>
> `Model Name`：Find the model name you need to call in [Models & Pricing](/Aggregation/Model-Price.mdx) and copy it

## Process

<Steps>
    <Step title="Step 1">
        <Card title="Click Settings" img="/images/API/cursor/step-1.jpg">
        </Card>
    </Step>
    <Step title="Step 2">
        <Card title="Change Key Settings" img="/images/API/cursor/step-2.jpg">
          Scroll down in `Models` on the left to find `OpenAI API Key` and enter `API Key` and `OpenAI Base URL`
          After setting, click `Verify`. If there are no errors, the connection is successful
        </Card>
    </Step>
</Steps>

<Warning>
For non-OpenAI models, you need to change the model name and add it for use
</Warning>

<Card title="Change Model Compatible Name" img="/images/API/cursor/tips.jpg">
    First click `Add Model`, and when adding a model, you need to add `liao-` before the model name
    For example, `grok-3` should be changed to `liao-grok-3`
</Card>
