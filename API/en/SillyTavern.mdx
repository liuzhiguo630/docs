## Preparation

> `Custom Endpoint`：https://ai.liaobots.work/v1
>
>
> `Custom API Key`：Your Login Code
>
>
> `Model Name`：Find the model name you need to call in [Models & Pricing](/Aggregation/Model-Price.mdx) and copy it

## Process

<Steps>
    <Step title="Step 1">
        <Card title="Change Key Settings" img="/images/API/SillyTavern/step-1.jpg">
          Click the plug icon at the top and fill in `Custom Endpoint`, `Custom API Key`, `Model Name`
        </Card>
    </Step>
    <Step title="Step 2">
        <Card title="Test Connection" img="/images/API/SillyTavern/step-2.jpg">
          Click the `Connect` button in the bottom left corner, then click `Send Test Message`
        </Card>
    </Step>
</Steps>
