## Preparation

Claude Code billing method is synchronized with official. For price ratio, please refer to [Models & Pricing](/Aggregation/Model-Price)

```json settings.json {2,3}
{
  "env": {
    "ANTHROPIC_AUTH_TOKEN": "Your Login Code",
    "ANTHROPIC_BASE_URL": "https://cc.liaobots.work/claudecode"
  },
  "permissions": {
    "allow": [],
    "deny": []
  }
}
```

## Process

<Steps>
  <Step title="Step 1">
    <Card title="Verify Node.js Installation" img="/images/API/ClaudeCode/step-1.png">
      Open terminal and enter `node --version` and `npm --version` to verify Node.js installation
      If you see a version number like `v22.16.0`, proceed to the next step. Otherwise, please download from the official website https://nodejs.org/en/download/
    </Card>
  </Step>
  <Step title="Step 2">
    <Card title="Install ClaudeCode" img="/images/API/ClaudeCode/step-2.png">
      Open terminal and enter `npm install -g @anthropic-ai/claude-code` to install ClaudeCode globally
      If you see the prompt as shown in the image, it means installation is successful
    </Card>
  </Step>
  <Step title="Step 3">
    <Card title="Create Configuration File" img="/images/API/ClaudeCode/step-3.png">
      Create a `.claude.json` file (note: `.claude.json`, not `.claude` folder) in the user directory, add the following content:
      ```json
      {
        "hasCompletedOnboarding": true
      }
      ```
    </Card>
  </Step>
  <Step title="Step 4">
    <Card title="Edit Configuration File">
      Then run `cd ~/.claude` and `vim settings.json` commands in sequence to enter configuration file editing (depending on your operating system, vim might be vi, etc.). Copy and change the above configuration settings and paste them in, then press the `Esc` key and enter `:wq` command to save and exit
      After successful save, ClaudeCode can start working
    </Card>
  </Step>
  <Step title="Step 5">
    <Card title="Start ClaudeCode" img="/images/API/ClaudeCode/step-4.png">
      Use the cd command to enter the project folder you want to edit, enter `claude` to start ClaudeCode, and then you can start chatting
    </Card>
  </Step>
</Steps>
