## Preparation

> `API Host`：https://ai.liaobots.work/v1
>
>
> `API Key`：Your Login Code
>
>
> `Model`：Find the model name you need to call in [Models & Pricing](/introduction.mdx) and copy it


## Process
<Steps>
    <Step title="Step 1">
        <Card title="Click Settings" img="/images/API/chatbox/step-1.jpg">
          Click Settings in the bottom left corner
        </Card>
    </Step>
    <Step title="Step 2">
    <Card title="Add Custom Model Provider" img="/images/API/chatbox/step-2.jpg">
      Click Add Custom Provider in Model Provider
    </Card>
    </Step>
    <Step title="Step 3">
    <Card title="Change Key Settings" img="/images/API/chatbox/step-3.jpg">
      Scroll down to change `API Host`, `API Key`, and `Model`
    </Card>
    </Step>
</Steps>
