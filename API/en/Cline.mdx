![Cline](/images/API/cline/cline-icon.jpg)

Search for cline in VSCode and install

Official website: [https://code.visualstudio.com/](https://code.visualstudio.com/)

## Preparation:

> `Base URL`：https://ai.liaobots.work/v1
>
>
> `API Key`：Your Login Code
>
>
> `Model ID`：Find the model name you need to call in [Models & Pricing](/Aggregation/Model-Price.mdx) and copy it

## Process

<Steps>
    <Step title="Step 1">
        <Card title="Choose to Use Your Own API" img="/images/API/cline/step-1.jpg">
          Select `Use your own API Key`
        </Card>
    </Step>
    <Step title="Step 2">
    <Card title="Change Custom Model Provider to OpenAI Compatible" img="/images/API/cline/step-2.jpg">
      Select `OpenAI Compatible` in `API Provider`
    </Card>
    </Step>
    <Step title="Step 3">
    <Card title="Change Key Settings" img="/images/API/cline/step-3.jpg">
      Fill in `BASE URL`, `API Key`, and `Model ID`
    </Card>
    </Step>
    <Step title="Step 4">
    <Card title="Test Connection" img="/images/API/cline/step-4.jpg">
      Click `plan` in the bottom right corner and send a test message
    </Card>
    </Step>
</Steps>
