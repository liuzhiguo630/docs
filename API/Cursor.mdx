安装 Cursor

官网地址: [https://www.cursor.com/cn](https://www.cursor.com/cn)

## 准备工作

> `OpenAI Base URL`：https://ai.liaobots.work/v1
>
>
> `API Key`：您的登陆码
>
>
> `模型名称`：在 [模型名与价格](/Aggregation/Model-Price.mdx) 找到您需要调用的模型名并复制下来

## 流程

<Steps>
    <Step title="第一步">
        <Card title="点击设置" img="/images/API/cursor/step-1.jpg">
        </Card>
    </Step>
    <Step title="第二步">
        <Card title="更改关键设置" img="/images/API/cursor/step-2.jpg">
          在左侧 `Models` 里下滑找到 `OpenAI API Key` 并输入 `API Key` 与 `OpenAI Base URL`
          设置完成后点击 `Verify` 无报错即为连接成功
        </Card>
    </Step>
</Steps>

<Warning>
对于非 OpenAI 模型需更改模型名并添加使用
</Warning>

<Card title="更改模型兼容名称" img="/images/API/cursor/tips.jpg">
    先点击 `Add Model` ，且添加模型需在模型名前加 `liao-`
    例如 `grok-3` 应改为 `liao-grok-3`
</Card>
