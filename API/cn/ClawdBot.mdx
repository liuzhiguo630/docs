## 准备工作

参照[官方文档](https://docs.molt.bot/start/getting-started)安装 Moltbot（旧名 Clawdbot）。支持macOS, Windows (WSL2), Linux。

文档全英文太难理解？使用 [Liao 接入沉浸式翻译](/API/cn/ImmersiveTranslation)

各模型价格比例请参考 [模型与价格](/Aggregation/Model-Price)

## 找到 Moltbot 配置文件

**macOS / Linux / WSL2:**
`~/.moltbot/moltbot.json` 或 `~/.clawdbot/clawdbot.json`

## 在 Moltbot 中切换当前使用模型

在上一步配置完成后，可使用以下命令格式切换模型（命令仅供参考，您只需要运行您需要的模型）：

```shell
moltbot models set liao/glm-4.7
moltbot models set liao-claude/claude-sonnet-4-5-20250929
moltbot models set liao-codex/gpt-5.2
moltbot models set liao-gemini/gemini-3-pro-preview
```
