## 准备工作

> `自定义端点`：https://ai.liaobots.work/v1
>
> `自定义 API 密钥`：您的登陆码
>
> `模型名`：在 [模型名与价格](/Aggregation/Model-Price.mdx) 找到您需要调用的模型名并复制下来
